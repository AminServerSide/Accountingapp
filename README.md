# Simple Accounting App

A lightweight and user-friendly accounting application built with C# and Windows Forms. This application allows users to manage financial transactions, track balances, and generate simple reports. Perfect for small-scale personal or business accounting needs.

## Features

- Add, edit, and delete transactions.
- Categorize transactions (e.g., income, expenses).
- View transaction history with filters.
- Generate balance summaries.
- Export transaction data to CSV.


## Technologies Used

- **Programming Language:** C#
- **Framework:** .NET Framework
- **UI:** Windows Forms
